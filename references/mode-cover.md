# 封面模式（cover）

为公众号/小红书文章生成封面配图。自动提取内容核心隐喻，从 prompts.yaml 选择高转化率 prompt 模板。

## 一、11 种封面风格速查

所有 prompt 模板统一存储在 `config/prompts.yaml` 的 `cover` 节。以下是速查表：

| # | 风格 ID | 名称 | 比例 | 关键词 | 视觉特征 |
|---|---------|------|------|--------|---------|
| 1 | visual_note | 视觉笔记 | 4:3 | 视觉笔记/手绘笔记/notion | 扁平信息图+手绘铅笔质感，白底蓝橙点缀 |
| 2 | hand_drawn | 手绘插画 | 3:4 | 手绘/插画/可爱/小红书 | 手绘彩铅风，暖橘赤陶，可爱比例 |
| 3 | mono_bw | 极简黑白 | 4:3 | 黑白/极简/高级/克制 | 纯黑白，大量留白，理性克制 |
| 4 | business | 商务资讯 | 4:3 | 商务/专业/都市/资讯 | 城市俯拍，深蓝灰白，专业感 |
| 5 | cyberpunk | 赛博科技 | 4:3 | 赛博/科技/霓虹/未来 | 深蓝黑底，霓虹发光，科技沉浸 |
| 6 | collage | 拼贴海报 | 4:3 | 拼贴/贴纸/海报/活泼 | 撕纸拼贴，荧光笔，贴纸装饰 |
| 7 | tropical | 热带雨林 | 4:3 | 热带/雨林/自然/探索 | 雨林背景，绿棕金自然色调 |
| 8 | illustration | 手绘插画风 | 4:3 | 手绘/书房/温暖/专注 | 温暖书房，手绘气泡，彩铅质感 |
| 9 | kawaii | 软萌可爱 | 3:4 | 可爱/萌/粉嫩/少女 | 粉色梦幻，卡通贴纸，软萌动物 |
| 10 | warm_healing | 温暖治愈 | 4:3 | 治愈/温暖/宁静/柔和 | 金色草原，樱花树，猫和奶茶 |
| 11 | watercolor | 水彩写意 | 4:3 | 水彩/写意/文艺/淡雅 | 水彩晕染，山峦天空，艺术感 |

> 来源：1-2 为原有模板；3-11 来自阿昆《一键搞定公众号封面》9 种风格，适配 Gemini 英文 prompt。

**默认风格**：用户未指定时 → visual_note（公众号 4:3）/ hand_drawn（小红书 3:4）。

## 二、封面类型识别

| 触发词 | 封面类型 | 比例 | 默认风格 |
|--------|---------|------|---------|
| 公众号 / 公众号封面 / 无明确平台（默认） | 公众号封面 | 4:3 | visual_note |
| 小红书 / 小红书封面 | 小红书封面 | 3:4 | hand_drawn |

**同时生成两种封面**：当用户说"封面"且未指定平台时，默认同时生成公众号和小红书两个版本。

## 三、核心隐喻提取

从用户的标题/主题中提取视觉隐喻。关键原则：

1. **具象化** — 抽象概念转为可画的东西
   - "AI技能分类学" → 一棵技能树 / 一个工具箱
   - "独立开发者" → 一个戴眼镜的人在工位上
   - "写作流水线" → 传送带上的纸 / 齿轮驱动的笔

2. **单一焦点** — 每张封面只有一个核心视觉元素

3. **隐喻映射表**：

| 主题领域 | 推荐隐喻 |
|---------|---------|
| AI/技术 | 大脑、电路、齿轮、机器人、灯泡、望远镜 |
| 写作/内容 | 笔、书、传送带、齿轮、放大镜 |
| 商业/创业 | 火箭、山峰、桥梁、指南针、地图 |
| 学习/知识 | 树、金字塔、书架、灯泡、拼图 |
| 个人成长 | 阶梯、种子→大树、蝴蝶、镜子 |

## 四、工作流

### Step 1：分析内容 → 提取核心隐喻

- 从用户输入的标题/主题中提取核心概念
- 将抽象概念映射为具体视觉隐喻
- 输出：一个清晰的英文视觉描述

### Step 2：选择封面类型和风格

- 根据触发词判断：公众号（4:3）/ 小红书（3:4）/ 两者都生成
- 扫描用户描述中的风格关键词，匹配上表中的风格
- 无明确暗示时，使用默认风格

### Step 3：从 prompts.yaml 获取 prompt

- 读取 `config/prompts.yaml` 的 `cover` 节
- 将核心隐喻代入对应风格的 prompt 模板，替换 `{METAPHOR}`

### Step 4：调用 generate.py

```bash
# 快捷方式（推荐，--style 自动读取比例和模板）：
python scripts/generate.py \
  --config config/engine.json \
  --style cyberpunk --prompt "a brain connected to circuit boards" \
  --out "output/主题/cover_gzh.jpg"

# 手动拼 prompt（无 --style）：
python scripts/generate.py \
  --config config/engine.json \
  --prompt "[从 prompts.yaml 读取并替换 {METAPHOR} 后的完整 prompt]" \
  --out "output/主题/cover_gzh.jpg" \
  --aspect-ratio "4:3"
```

### Step 5：报告输出 + 飞书推送

```
封面生成完成
- 风格：[风格名]（[风格ID]）
- 公众号封面：[绝对路径]（4:3）
- 小红书封面：[绝对路径]（3:4）
```

**自动推送飞书**（生成完成后自动执行，无需用户确认）：
```bash
lark-cli im +messages-send --chat-id oc_77e72a8fa29356bf8942ac739bbd6dd5 --as bot --file [图片路径]
```

## 五、注意事项

1. **Prompt 不要加文字描述** — 封面图是纯视觉，文字在发布时叠加（文章 9 风格除外，其自带排版）
2. **隐喻优先于写实** — 用隐喻/符号而不是真人/实景
3. **避免版权风险** — 不用品牌 logo、真人肖像、受版权保护的形象
4. **输出目录** — 默认写入 `output/` 下按主题命名的子目录
